Acceso Administrador

Para acceder, crea un usuario en Supabase:

  1. Ve a tu proyecto Supabase Dashboard → Authentication → Users
  2. Click "Add user" → "Create new user"
  3. Ingresa email, contraseña y confirma el email.

Configuración Inicial (Requerido para la gestión de categorías):

  1. Crear Tabla de Categorías: En Table Editor, crea una tabla llamada categorias con columnas: id (bigint, Primary Key, auto-incremental), nombre (text), slug (varchar).
  2. Actualizar Tabla de Productos: Borra la columna categoria (varchar) de tu tabla productos. Crea una nueva columna categoria_id (bigint) y establece una foreign key que referencie a categorias(id).
  3. Permitir Acceso Público: Ve a Authentication → Policies y crea nuevas políticas para categorias y configuracion para permitir la lectura pública (SELECT) a todos los usuarios (rol: anon).
